## 📦 **Download & Install**

### 🎯 **Quick Start**
1. **Download** the latest release from the [Releases](https://github.com/allano0/forou_pos_system_releases/releases) page
2. **Run** `Forou-POS-{version}-setup.exe`
3. **Follow** the installation wizard
4. **Launch** and start selling! 💰

### 📋 **System Requirements**
- **OS**: Windows 10/11 (64-bit)
- **RAM**: 4GB minimum, 8GB recommended
- **Storage**: 500MB free space
- **Network**: Internet connection for data sync

## 🔧 **Installation Guide**

### **Step 1: Download**
```bash
# Download the latest release
wget https://github.com/allano0/forou_pos_system_releases/releases/latest/download/Forou-POS-1.0.4-setup.exe
```

### **Step 2: Install**
1. **Right-click** the downloaded file
2. **Select** "Run as administrator"
3. **Follow** the installation prompts
4. **Choose** installation directory
5. **Complete** the setup

### **Step 3: First Launch**
1. **Launch** Forou POS from Start Menu or Desktop
2. **Sign in** with your credentials
3. **Configure** your business settings
4. **Start** processing sales!
